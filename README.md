# Wallet App

A simple credit/debit wallet system built with Java and Spring Boot.

## Features
- View current balance
- Add money
- Withdraw money
- View transaction history (optional)

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- H2 in-memory database

## How to Run
1. Clone the repository
2. Run the application using your IDE or `mvn spring-boot:run`

## Test Cases
Located in `src/test/java/com/walletapp`
